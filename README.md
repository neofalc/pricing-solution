## Ноутбук

`homework3_pricing_solution.ipynb` содержит полный ход решения:

- загрузка и предварительная обработка данных;
- разведочный анализ данных;
- статистическое моделирование;
- обучение модели машинного обучения;
- оценка качества модели.

## Зависимости

Для запуска ноутбука нужен Python и основные библиотеки:

- numpy
- pandas
- statsmodels
- matplotlib
- scikit-learn

Установить зависимости можно командой:

```bash
pip install numpy pandas statsmodels matplotlib scikit-learn jupyter
```
